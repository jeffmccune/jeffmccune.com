--- layout: post title: Gap - MacOSX Universal Binary Packages tags: - Mac OS X - System Administration status: publish type: post published: true meta: {} --- I received a request to deploy Gap (Groups, Algorithms, Programming - a System for Computational Discrete Algebra). The installation process is a bit involved, so these packages might help, as they work well with Apple Remote Desktop. This package places gap at /usr/local/gap4r4 and places the executable at /usr/local/bin/gap The package does not modify the PATH, so you'll have to manage that yourself. The package is: Gap-4.4.9-1.dmg
